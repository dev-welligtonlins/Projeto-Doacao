CONHEÇA NOSSO PROJETO

A plataforma é um site de doações para todo Brasil. Mas você sabe como essa história começou?

Desenvolvimento Do Projeto

O projeto DoAção existe pela parceria entre a Universidade Federal do Acre e a Motorola através do projeto de capacitação de desenvolvimento Full-Stack Web Academy. Nosso time busca aumentar as doações recebidas pelas instituições através da publicação das suas campanhas em um espaço reservado, seguro e de fácil acesso. Os doadores conseguirão acompanhar as instituições e os resultados das campanhas.

Nosso objetivo é contribuir com a sociedade, intermediando doações para aqueles que necessitam de ajuda.

Missão

Contribuir para que as pessoas em situação de exclusão, possam receber ajuda através de um novo projeto.

Visão

Ser a plataforma que intermedia doações por todo Brasil, alcançando todos!
